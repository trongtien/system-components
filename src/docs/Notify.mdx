# Notify

Display a notification message globally.

## When To Use

To display a notification message at any of the four corners of the viewport. Typically it can be used in the following cases:

- A notification with complex content.
- A notification providing a feedback based on the user interaction. Or it may show some details about upcoming steps the user may have to follow.
- A notification that is pushed by the application.

## Usage

### HTML (Web Component)

```html
<system-notify id="notify" placement="topRight"></system-notify>

<script>
  const notify = document.getElementById("notify");

  // Show info notification
  notify.info("This is an info message");

  // Show success notification
  notify.success("Operation completed successfully!");

  // Show error notification
  notify.error("Something went wrong!");

  // Show warning notification
  notify.warning("Please be careful!");

  // Show with custom config
  notify.show({
    message: "Custom notification",
    type: "info",
    description: "This is a detailed description",
    duration: 5000,
  });

  // Close all notifications
  notify.closeAll();
</script>
```

### React

```jsx
import { useRef } from "react";

function App() {
  const notifyRef = useRef(null);

  const showNotification = () => {
    notifyRef.current?.success("Hello from React!");
  };

  return (
    <>
      <system-notify ref={notifyRef} placement="topRight" />
      <button onClick={showNotification}>Show Notification</button>
    </>
  );
}
```

### Vue

```vue
<template>
  <system-notify ref="notify" placement="topRight" />
  <button @click="showNotification">Show Notification</button>
</template>

<script setup>
import { ref } from "vue";

const notify = ref(null);

const showNotification = () => {
  notify.value?.success("Hello from Vue!");
};
</script>
```

## API

<Controls />

### NotifyProps

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| placement | Position of notification | `topRight` `topLeft` `bottomRight` `bottomLeft` `topCenter` `bottomCenter` | `topRight` |
| maxCount | Max notifications to show simultaneously | `number` | `5` |
| style | Custom style | `string` `CSSProperties` | - |

### Methods

| Method | Description | Parameters |
|--------|-------------|------------|
| show | Show a notification | `(config: { message, type?, description?, duration? }) => string` |
| info | Show info notification | `(message: string, duration?: number) => string` |
| success | Show success notification | `(message: string, duration?: number) => string` |
| error | Show error notification | `(message: string, duration?: number) => string` |
| warning | Show warning notification | `(message: string, duration?: number) => string` |
| close | Close a specific notification | `(id: string) => void` |
| closeAll | Close all notifications | `() => void` |

### NotifyConfig

| Property | Description | Type | Default |
|----------|-------------|------|---------|
| message | Content of notification | `string` | - |
| type | Type of notification | `info` `success` `error` `warning` | `info` |
| description | Additional description | `string` | - |
| duration | Time before auto-close (0 = never close) | `number` | `3000` |
